课程设计.课程设计文档生成器 · 以「数据库原理与应用」为示范

数据驱动的课程设计文档生成平台。把教学大纲、教案、考核方案结构化为 JSON，按 OBE 教学理念、学情分层、学时分档规则，一键渲染为标准化的课程设计文档。

course_info.json
{
"course_name": "数据库原理与应用",
"course_code": "04042603a",
"credits": 3,
"total_hours": 48,
"student_level": "undergraduate",
"assessment": { "平时": 20, "实验": 20, "期末": 60 }
}

⚠ 此为 v0.1 存档版 · 后续版本见 index_v0.2_* 及更高版本

核心能力
01

Data-Driven

课程信息、教学单元、思政案例库 — 所有内容写成 JSON。换课程只需替换数据文件。

02

Rule Engine

按学时自动分档（≥48/32-47/≤31），按学情自动分层，按 OBE 矩阵映射毕业要求。

03

MD + Docx

Markdown 预览可编辑，Docx 可直接提交教务处。
